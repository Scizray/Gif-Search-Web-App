<h2>🧐 Features</h2>

Here're some of the project's best features:

*   Authentication System to Signup and Login
*   Search GIFs on Giphy
*   Hot Search - Gifs Updated Dynamically

  
  
<h2>💻 Built with</h2>

Technologies used in the project:

*   Next.js
*   Firebase Authentication
*   Giphy API
